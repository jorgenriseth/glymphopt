# glymphopt
This repository contains code for investigating inverse modelling of parameters for global solute transport models in the brain.


Source code is available at < https://github.com/jorgenriseth/glymphopt>

## Getting started
We provide a pre-build Docker image which can be used to run the the code in this repository. First thing you need to do is to ensure that you have [docker installed](https://docs.docker.com/get-docker/).

To start an interactive docker container you can execute the following command

```bash
docker run --rm -it ghcr.io/jorgenriseth/glymphopt:latest
```
